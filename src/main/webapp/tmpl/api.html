API

GET http://{{host}}/rest/bidEngine/current
  • @Produces : 200 response with an application/json response object {"itemName":"name","value":10,"timeToLive":3945}
  • @Produces : 404 response if no more bid offer is available
POST http://{{host}}/rest/bidEngine/bid
  • @Consumes : application/json request object {"itemName":"name","value":10} with an Authorization: {key} header
  • @Produces : 200 response with an application/json response object {"itemName":"itemName","value":10,"timeToLive":3945}
  • @Produces : 4XX response if bid is rejected with a text/plain response message
POST http://{{host}}/rest/bidEngine/offer
  • @Consumes : application/json request object {"itemName":"name","value":10} with an Authorization: {key} header
  • @Produces : 200 response with an empty body
  • @Produces : 4XX response if offer is rejected with a text/plain response message
WebSocket ws://{{host}}/socket/bidEngine/{key}
  • @Consumes : application/json object {"bid":{"itemName":"name","value":10}}
  • @Consumes : application/json object {"offer":{"itemName":"name","value":10}}
  • @Produces : application/json object {"bidOffer":{"itemName":"name","value":10,"timeToLive":3945}}
  • @Produces : application/json object {"info":"message"}
  • @Produces : application/json object {"error":"message"}
